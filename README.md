# Student Marks Analyzer

A Python data analysis project that reads student marks from a CSV file, computes total and average scores, identifies the highest and lowest scorers, generates statistical summaries, and visualizes each student’s marks using bar charts.

## Features
- Reads student marks from a CSV file
- Calculates Total and Average marks per student
- Identifies highest and lowest scoring students
- Displays statistical summary (mean, min, max, etc.)
- Visualizes subject-wise marks for each student using bar charts

## Technologies Used
- Python
- Pandas – data manipulation and analysis
- Matplotlib – data visualization
- CSV file handling

## How to Run
- Install dependencies
   pip install pandas matplotlib
- Run the program
   python main.py
